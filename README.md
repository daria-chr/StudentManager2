# StudentManager2
Назначение программы
приложение для управления списком студентов с использованием базы данных SQLite

Реализованные функции
Добавление новых записей о студентах
Удаление выбранных записей из списка
Отображение списка всех студентов в таблице
Автоматическое обновление интерфейса после изменений

Запуск проекта
Склонируйте репозиторий на локальный компьютер
Откройте файл решения StudentManager.sln в Visual Studio
Восстановите необходимые NuGet пакеты
Запустите проект комбинацией клавиш F5 или через меню Debug
